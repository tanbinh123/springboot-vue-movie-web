### 存在问题

* java项目打包后会导致上传图片的功能开发环境和生产环境路径不同导致无法正常上传服务器，因此需考虑到这一点
* 未完成用户登录的功能，本项目主要练习springboot实现crud等基本操作
* 后端解决了跨域任何网站都可以进行访问，存在安全隐患
* 数据库设计和SQL语句联表查询可能存在问题可能导致查询缓慢
* 无访问权限管理，密码未加密处理
